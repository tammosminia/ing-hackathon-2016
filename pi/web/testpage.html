Push API

Install app on your android phone

Download the apk.
Get it on your phone. It's the easiest to mail it to your gmail address.
Install the apk. You'll need to accept non-trusted applications for this.
Start the app. It will display the registration id. Copy that.
Now you're ready for receiving messages. Put your registration Id below together with your message and push send.

This page does not work well on internet explorer 10.

This works with the TestPush app of 24 feb 2016. TestPush will print the version on standup. If it doesn't, you're using an old version.
